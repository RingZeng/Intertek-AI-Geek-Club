📤 如何上传文件？

由于GitHub Pages是静态页面，文件上传需要通过GitHub仓库：

  1. 访问 GitHub仓库
  2. 点击"Add file" → "Upload files"
  3. 拖拽文件到指定区域
  4. 提交后文件将出现在列表中
